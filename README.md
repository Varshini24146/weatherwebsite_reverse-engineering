A reverse-engineered Weather Forecast Web Application that provides real-time weather updates, forecast visualization, air quality insights, and smart user notifications using API-driven data and dynamic frontend rendering.

Features

City-based real-time weather search

Current weather analytics (temperature, humidity, wind)

Hourly and multi-day forecast graphs

Smart health, travel, and activity recommendations

Air Quality and Allergy information modules

Responsive modern UI

Tech Stack

HTML | CSS | JavaScript | Chart.js | WeatherAPI

Run

Clone the repository and open index.html in any browser.
